# 💾 Installation
1. Clone the repo
2. Set up environment variables 
3.
```sh
pnpm install
```
4.
```sh
pnpm build
```
5.
```sh
pnpm start 3000
```

# ⛏️ Built Using
- Next.js - Web framework
- Redux - State management
- Prisma - ORM
- PostgreSQL - Database
- Chakra UI - For building the UI
- pnpm - Packaga manager
